# RiseMemo

Maintenance your project with memo. Based with React Typescript and Material UI

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

## Library

-React MaterialUI
-React DnD
-immutability-helper - for update state easily
